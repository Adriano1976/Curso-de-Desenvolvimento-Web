Serviços de Intranet, Extranet, Wap e Voip

1. Qual a finalidade da Internet 2?

A Internet 2 tem o propósito de desenvolver e testar o funcionamento de aplicações e tecnologias inovadoras para as redes, ainda não disponíveis para a Internet normal. Veja neste vídeo, a finalidade da internet 2 e seus benefícios.


2. Quais as principais características da Intranet?

A Intranet tem como características principais: funcionar como a Internet, mas no âmbito de uma empresa. Veja neste vídeo, disponibilizado por P1R@T3, fala das suas principais características.


3. Quando são criadas as Extranets?

Quando é necessário dar acesso a Intranet para uma pessoa que não é empregado da empresa ou para outra empresa. Neste vídeo, disponibilizado por P1R@T3, fala sobre o que é e qual a finalidade das extranets.


4. O que são Web services? Qual a sua finalidade?

São serviços oferecidos via Web que ficam disponíveis para os usuários. A finalidade dos Web Services é integrar diferentes equipamentos e softwares que acessam a Web, para que possam se comunicar e utilizar os serviços oferecidos de uma forma padronizada. Veja neste vídeo explicativo, disponibilizado pelo canal Moura Informática, fala como funciona e o que são web services.


5. O que é WAP?

WAP é um protocolo que especifica como dispositivos móveis, principalmente telefones celulares, podem acessar a Internet. Neste vídeo explicativo de Elis Monteiro, disponibilizado pelo canal Fã Clube da Elis Monteiro, fala com detalhes o que é wap e como usar.


6. Qual a ideia por trás do VoIP (Voz sobre IP)?

A idéia do VoIP é utilizar a infra-estrutura das redes TCP-IP conectadas a Internet para conduzir o tráfego de voz (ligações telefônicas). Veja neste vídeo, feito pelo Olhar Digital, a explicação sobre VoIP e sua definição.


7. Qual a diferença entre intranet e extranet?

Intranet é uma rede dentro de uma organização que usa tecnologias internet para permitir que os usuários encontre, usem e compartilhem documentos e páginas Web. Corporações usam intranets para comunicarse com seus empregados. Neste vídeo, produzido por Jonatas de O. Menezes e disponibilizado no canal de Lucas Pereira dos Santos, explica com mais detalhes sobre suas diferenças.
