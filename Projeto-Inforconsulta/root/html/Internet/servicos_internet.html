Serviços de Internet

1. Qual a finalidade de serviço de FTP?

O serviço FTP permite que os usuários localizem e movam arquivos de um lugar para outro da rede. Nesta vídeo aula, produzido pela Classe A Cursos e disponibilizado no canal de Henrique Polo, ensina quais as vantagens de se ter um FTP e como configurar e instalar um Servidor FTP em sua máquina.


2. Como funciona o serviço de correio eletrônico (e-mail) e como é composto o endereço de um usuário desse serviço?

O serviço de correio eletrônico funciona como as caixas postais do correio. Quando alguém deseja enviar uma mensagem para outra pessoa, ela utiliza o endereço eletrônico dessa pessoa para encaminhar a mensagem. Essa mensagem será enviada para uma caixa postal (no servidor de correio eletrônico do provedor de acessos) e ficará lá até que o destinatário da mensagem leia o conteúdo de sua caixa postal. O endereço eletrônico é formado por uma identificação do usuário, seguido do símbolo @ (arroba) e depois uma identificação do servidor de correio. Dessa forma os endereços eletrônicos da Internet são únicos em nível mundial. Nesta vídeo aula, produzido e dispo-nibilizado no canal DegrauCultural, Professor Luciano Antunes nos dá uma dica sobre correio eletrônico. Diz como funciona o correio eletrônico e qual detalhe ficar atento quando esse assunto cair na prova.


3. Como é o funcionamento das páginas Web?

As páginas Web podem ser estáticas ou dinâmicas. As páginas estáticas são aquelas criadas para que apresentem sempre a mesma aparência, textos, imagens e hyperlinks. As páginas dinâmicas são aquelas criadas a partir da ação realizada pelo usuário. Veja neste vídeo explicativo, como funciona as páginas web e como é feito da transmissão de dado. Esse vídeo foi produzido e disponibilizado no canal Site Em Angola .Webdesign Angola.


4. Qual a função de um hub em uma LAN?

As páginas Web são criadas utilizando a linguagem de programação HTML (HyperText Markup Language).

5. Qual a composição de um URL?

Um URL é composto por:
 Protocolo;
 Tipo de servidor;
 Nome de domínio;
 Nome do diretório;
 Nome do arquivo;
Vejamos o que este vídeo explicativo sobre a linguagem HTML, produzido e disponibilizado pelo canal StudyClassOficial, do porque dessa linguagem ser a essência das páginas web.


6. Para que serve um browser? Como é possível estender a sua funcionalidade?

Browser é uma aplicação cliente que permite ao usuário acessar os dados de uma página Web. Para estender a funcionalidade dos browsers, ou seja, acessar conteúdos diferentes daqueles criados em HTML, é possível realizar a instalação de plugins. Plug-ins são módulos de código instalados no browser, que permitem a apresentação de arquivos com formatos diferentes do HTML. Veja neste vídeo explicativo, feito pelo professor Rodrigo Schaeffer e disponibilizado no seu canal de Rodrigo Schaeffer, sobre o assunto.

8. O que são Weblogs?

Weblog (Blog) é uma aplicação Web que está presente em determinados sites que recebem informações colocadas periodicamente e na ordem cronológica inversa. As informações que constam dos blogs estão disponíveis para qualquer usuário da Web que deseje acessá-las. Nesta vídeo-aula, disponibilizado no canal konfideconsultoria, fala sobre isso.


9. Como funciona a troca de arquivos P2P (peer-to-peer)?

A troca de arquivos P2P acontece de computador para computador sem a intervenção de um servidor, utilizando diretamente a Internet como meio de comunicação. Neste vídeo explicativo, produzido e disponibilizado no canal pelo TecMundo, fala sobre sobre o assunto.


Que cuidados devem ser tornados na criação de uma página Web?

Caso a página Web a ser criada for estática, a utilização do HTML é suficiente, mas se houver a necessidade de páginas dinâmicas ou com acesso a bancos de dados, então será necessário utilizar algumas linguagens ou aplicativos que permitam esse comportamento. Outra preocupação é com o local onde os arquivos da página serão armazenados (hospedados). Além disso, é importante se preocupar com a necessidade, ou não, da aquisição de um nome de domínio.
