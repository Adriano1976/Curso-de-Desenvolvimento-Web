Sistemas Operacionais de Rede

1. Como podem ser classificados os sistemas operacionais de rede?

Os sistemas operacionais de rede podem ser classificados como sistema operacional cliente ou sistema operacional servidor. A integração entre eles ocorre quando uma porção cliente do sistema operacional de rede é instalada nos computadores dos usuários. Essa porção cliente se comunica com a porção servidora permitindo que o sistema operacional escolha se a operação deve ser realizada pelo sistema operacional cliente (de forma local) ou pelo sistema operacional servidor (em nível de rede). Entenda, neste vídeo feito e disponibilizado no canal Espiral, explica com detalhe, qual é o conceito de clientes e servidores em redes de computadores.


2. Que características deve apresentar um sistema operacional de rede?

Características de servidor, sistema de arquivos, confiabilidade, segurança, independência, facilidade de instalação e configuração e administração remota. Neste vídeo explica os tipos e as características dos sistemas operacionais de rede.


3. Quais são os serviços oferecidos pelos sistemas operacionais de rede?

Serviços de arquivos, serviços de impressão, serviços de mensagem e serviços de diretório.

4. Que características apresenta a arquitetura peer-to-peer?

Todas as estações de trabalho podem atuar como servidor, em algum momento do trabalho, portanto não existe um computador dedicado exclusivamente a essa tarefa. Neste vídeo disponibilizado no canal nazarenouberlandia, explica com detalhes do funcionamento de uma rede ponto-a-ponta.


5. Que características apresenta a arquitetura cliente / servidor?

Os computadores que estão conectados a esse tipo de rede podem operar exclusivamente de um modo: servidor ou cliente. O servidor é um computador mais potente e dedicado somente à tarefa de oferecer algum dos serviços apresentados anteriormente. O cliente é o computador utilizado pelos usuários e se conectam ao servidor por meio da rede. Os computadores cliente só conseguem se comunicar com os computadores servidores. Nesta vídeo aula, feito pelo professor Binho e disponibilizado em seu canal Professor Binho, explica os princípios e características da arquitetura cliente/servidor.


6. Qual a vantagem de utilizar servidores dedicados?

A utilização de servidores dedicados permite que os serviços oferecidos sejam mais eficientes, pois a administração desse serviço é realizada por softwares específicos para essa tarefa. Neste vídeo, disponibilizado no canal wwwovhes, demonstra como funciona o servidor dedicado e qual as suas características de uma forma bem divertida. Observem com atenção essa apresentação.
