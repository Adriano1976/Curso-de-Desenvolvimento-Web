Redes Sem Fio

1. Qual a utilidade das redes sem fio?

Estabelecer comunicação sem a necessidade de conexão física entre as partes envolvidas, permitindo maior flexibilidade. Neste vídeo, feito pelo programa de televisão Olhar Digital, explica como funciona o sistema de rede sem fio e seus benefícios.


2. O que é bluetooth?

É uma especificação criada por um conjunto de empresas que descreve a transmissão sem fio de dados (por meio sinais de rádio) entre telefones móveis, computadores e outros equipamentos. Essa tecnologia requer que um chip transceiver (transmissor e receptor) de baixa potência seja instalado em cada dispositivo envolvido na comunicação. Neste vídeo, feito pelo programa de televisão Olhar Digital, explica como funciona o bluetooth e seus benefícios.


3. O que são piconets?

É uma configuração de Bluetooth que utiliza um nó central mestre com até sete nós escravos, dentro de um raio de 10 metros.

4. Em uma scatternet, como um dispositivo de uma piconet pode se comunicar com um dispositivo de outra piconet?

Um nó escravo de uma piconet se comunica com o nó central dessa mesma piconet, enviando o pacote de dados para ele. O nó central envia o pacote de dados para um nó escravo que serve como ponte entre duas piconets (o nó é escravo dos nós mestre das duas piconets). Esse nó escravo (ponte) envia o pacote para o nó central e este para o nó destino do pacote de dados. Neste vídeo, disponibilizado pelo canal ComoTudoFunciona, fala sobre isso.


5. Quais são as configurações possíveis para uma LAN sem fio?

Com estação base (toda comunicação deve passar por um equipamento central) ou sem estação base (comunicação acontece de computador para computador).

6. Quais são os padrões da Wi-Fi?

Os padrões Wi-Fi são: 802.11b (HR-DSS) que opera na faixa de 2,4 GHz e atinge taxas de transmissão de até 11 Mbps e 802.11a (OFDM) que opera na faixa de 5 GHz e atinge taxas de transferência de até 54 Mbps. Neste vídeo, feito pelo programa de televisão Olhar Digital, explica os padrões da conexão sem fio de forma simplificada.
