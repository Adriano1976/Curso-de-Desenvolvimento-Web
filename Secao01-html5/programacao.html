Tipos de Linguagem de Programação

Programação Procedural
PHP / PYTHON Está relacionado a instaurar rotinas ou passo-a-passo a ser seguido em um programa.
Programação Funcional
R Está relacionado ao conceito de funções da matemática, sendo muito usado no meio acadêmico.
Programação Orientação Objeto
C# / C++ / PYTHON Está relacionado a interação entre diversos tipos de classes.
Programação de Script
JAVASCRIPT Está relacionado a execução de um escript, que consiste em uma série de passos feitos de forma automatizada.
